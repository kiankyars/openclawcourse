# OpenClaw 3-Hour Course

A structured curriculum for a **3-hour** Free Code Camp–style course on [OpenClaw](https://docs.openclaw.ai): the messaging gateway that bridges WhatsApp, Telegram, Discord, iMessage (and more) to AI coding agents (Pi).

## Prerequisites

- **Node ≥22** (runtime requirement per [docs](https://docs.openclaw.ai))
- **Docker** (optional but recommended for sandboxing)
- One of: WhatsApp, Telegram, or Discord account for pairing
- Terminal comfort; no prior OpenClaw experience
