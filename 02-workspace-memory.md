# Workspace & Memory

## Key Concepts

- Config, credentials, sessions live under `~/.openclaw/`.

See: [Agent Workspace](https://docs.openclaw.ai/concepts/agent-workspace)

## Git backup

- Treat workspace as **private** memory; init git, add private remote, `.gitignore` secrets.

See: [Git backup](https://docs.openclaw.ai/concepts/agent-workspace#git-backup-recommended-private)
