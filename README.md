# OpenClaw 3-Hour Course

Structured curriculum for a **3-hour** Free Code Camp–style course on [OpenClaw](https://docs.openclaw.ai): the messaging gateway that bridges WhatsApp, Telegram, Discord, iMessage (and more) to AI coding agents (Pi).

---

## Course Overview

| Module | Title | Time | Docs anchor |
|--------|--------|------|-------------|
| [01](01-intro-architecture/README.md) | Intro & Architecture | 20 min | [OpenClaw](https://docs.openclaw.ai), [Architecture](https://docs.openclaw.ai/concepts/architecture) |
| [02](02-install-first-run/README.md) | Installation & First Run | 25 min | [Getting Started](https://docs.openclaw.ai/start/getting-started), [Wizard](https://docs.openclaw.ai/start/wizard) |

---

## Prerequisites (Learners)

- **Node ≥22** (runtime requirement per [docs](https://docs.openclaw.ai))
- **Docker** (optional but recommended for sandboxing)
- One of: WhatsApp, Telegram, or Discord account for pairing
- Terminal comfort; no prior OpenClaw experience
